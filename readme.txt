この拡張機能を使うと、XをTwitter表示に戻すことができます。

以下、導入手順です。

0. images, js, manifest.json をダウンロードし、新規作成したフォルダ内に移動（「XtoTwitter」という名前で進めます）

1. Google Chromeを開き、拡張機能ボタンから「拡張機能を管理」へ。

2. デベロッパーモードをONにし、出てくる「パッケージ化されていない拡張機能を読み込む」をクリック。

3. XtoTwitterのフォルダを選択。（注: 中のファイルやフォルダではなく、正しく「XtoTwitter」を選択してください。）

4. 導入完了です。なお、既にTwitter（X）を開いていた場合はエラーが出ますが、その場合はTwitterをリロードしてください。動作に影響はありませんが、エラーは自動で消えませんので、気になるようでしたらエラーを削除してください。
